# Репозиторий КР1

Это репозиторий для вывода строки

## Установка

Исходный код программы находится в каталоге
```src```

Сборка приложения выполняется командой

```
gcc main.c
```

имеется репозиторий в гитхабе по [https://github.com/maingraph/TPMP-KR1/tree/main/src]
